# Web Search

Use the `web_search` tool to find current information on the internet.

## When to Use

- Current events and news
- Latest documentation or API references
- Fact checking
- Finding examples or tutorials
- Research on specific topics

## Examples

Search for current news:
```
web_search(query="latest AI developments 2024")
```

Find documentation:
```
web_search(query="React 19 new features documentation")
```

Research a topic:
```
web_search(query="best practices for TypeScript monorepos")
```

## Tips

- Be specific in your queries
- Use quotes for exact phrases
- Include year for time-sensitive topics
- Combine multiple keywords with spaces

## Response Format

The tool returns search results with:
- Title and URL
- Snippet/summary
- Source information

Use the information to provide accurate, up-to-date answers to the user.
